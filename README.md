<h1>  Python Script </h1>
<h3> Simple Keylogger Script </h3>

<h2> WHAT IS IT? </h2>
Created a Python Keylogger script.

<h2> HOW IT WORKS </h2>
Once the script has been executed, it will automatically create a log.txt file on the local machine and writes anything which is typed on the keyboard (all keystrokes) to the file created.



<h2> HOW TO USE IT </h2>
Run the script. <br>
<br>The script will automatically create the log.txt file in the same directory. <br>
<br> The script will save anything which is typed on the keyboard to the log.txt file until it's been stopped. <br>
